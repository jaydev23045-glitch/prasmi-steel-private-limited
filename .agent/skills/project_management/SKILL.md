# Project Management Skill

This skill provides instructions for managing the project's goals, updating `PLAN.md`, and ensuring the roadmap is followed.

## Instructions
1. Always check `PLAN.md` before starting a new task.
2. Update `PLAN.md` when a goal is completed.
3. Propose new goals based on project progress.
